# Lesson_OB05
Lesson_OB05 description

Задание
Разработать игру.

Примеры игр:

пинг-понг
змейка
тетрис
игра на выживание с постоянным передвижением, где на поле постоянно появляются “враги”, которых нельзя касаться


-------УПРАВЛЕНИЕ-------

Движение танков:
W / Стрелка вверх - движение вверх

S / Стрелка вниз - движение вниз

A / Стрелка влево - движение влево

D / Стрелка вправо - движение вправо

----Управление стрельбой----

Прошу обратить внимание включена инверсия при стрельба влево/вправо

Игрок 1: Пробел

Игрок 2: Правый Ctrl

-----ВНИМАНИЕ-----

При запуске игры возможно попадание танка на стену, чтобы исправить закройте игру и запустите заново!
